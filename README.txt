Animation Project 1 Basic skinned mesh animation
Compile and run in Visual Studio 2019 Community
for best performance use Release x64

Controls:
	- Mouse move
		rotate character left/right
		
	- W
		Move forward
	
	- Shift (while W is down)
		Run faster
	
	- S/D
		Strafe

	- Space
		Jump

	- E
		Punch

	- 1/2
		Change character
